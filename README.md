# Web-Task2-Data
This is a simple web application built with HTML, CSS, JavaScript, PHP, and MySQL.
It allows users to add a name and age through a form, view all records in a table, and toggle the user’s status between 0 and 1 instantly without reloading the page.
---
## Setup Instructions

1. Install and run a local server environment such as XAMPP.

2. Create a database named info in phpMyAdmin.

3. Inside that database, create a table named users with columns for id, name, age, and status.

4. Place all project files inside your server folder (for example: C:\xampp\htdocs\people-app\).

5. Open config.php and make sure the database name, username, and password match your local setup.

6. Start Apache and MySQL from your server control panel.

---
## How to Use


1. Open the project in your browser by going to http://localhost/people-app/index.php.


2. Enter a name and age, then click Submit to add a new record.


3. The new user will appear in the table immediately.


4. Click the Toggle button to switch the user’s status between 0 and 1.


5. All changes are saved directly to the database.
